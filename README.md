## Описание плагина

Данный плагин предназначен для скрытия файлов и папок, прописанных в `.gitignore`.

**Возможности**:

- Сокрытие файлов и папок, прописанных в `.gitignore`
- Показ скрытых файлов и папок

**Потенциальные улучшения**:

- Автоген `.gitignore`
